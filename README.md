# Amazon Image Renamer

Tool om Amazon-afbeeldingen automatisch te hernoemen op basis van:
- kleurvarianten
- ASIN's
- rollen (MAIN, SWCH, PT01, etc.)

## Gebruik

1. Voeg kleuren en bijbehorende ASIN's toe
2. Upload basisafbeeldingen
3. Kies per afbeelding een rol en kleur
4. Genereer ZIP en upload naar Amazon
